# IVORY ARCHIVE content guide

- Last audited: 2026-07-19
Primary data file: `app/briefings.ts`

## Project theme

IVORY ARCHIVE 是一份中文思想简报与长期阅读档案。它把当天值得停留的文化和社会新闻组织成可继续研究、讨论与创作的材料，而不是只提供快速标题摘要。

当前每期包含 5 则故事，并沿三条长期线索展开：

1. 艺术人文：艺术、设计、电影、音乐、文学、建筑、戏剧、历史、哲学、宗教与文化史。
2. 社会科学：心理学、社会学、人类学、语言学、传播学，也明确包括天文学与地理学，并关注科学背后的制度、基础设施和社会影响。
3. 女性主义：女性历史、性别、女性主义理论、健康、照护与权力结构。

分类允许交叉，例如当前已有“艺术人文 × 女性主义 × 艺术史”“社会科学 × 天文学 × 科研制度与基础设施”等复合标签。

## Target reader

- 想从文化新闻进入更深问题的普通读者。
- 关注艺术人文、社会科学与女性主义的读者。
- 需要选题、背景和创作角度的内容创作者。
- 希望保留来源、日期和事实卡，而不只阅读观点的人。

内容应让非专业读者能读懂，同时保留足够具体的数据、机构、日期和不确定性，让读者可以继续查证。

## Core editorial structure

首页：

- 最新一期的日期、期号、主题与简介
- “今日五则”卡片
- 分类筛选与搜索
- 历史日刊档案
- 三条长期主题范围

每期详情页：

- 期号、日期、主题、导言和主题标签
- 5 则故事索引
- 每则故事包含：

  | 字段 | 页面作用 | 内容边界 |
  | --- | --- | --- |
  | `title` / `summary` | 说明故事是什么 | 不能靠夸张替代事实 |
  | `happened` | 发生了什么 | 尽量是可由来源支持的事件与背景 |
  | `importance` | 为什么重要 | 可以解释，但要区分事实与分析 |
  | `creatorAngle` | 可创作角度 | 明确是建议或问题意识，不是已证实结论 |
  | `facts` | Fact File | 应具体、可核验、避免重复正文空话 |
  | `sourceName` / `sourceUrl` / `sourceDate` | 来源与时间 | 不得编造或用搜索结果页替代原始页面 |
  | `image` / `imageAlt` / `imageCredit` | 视觉证据和说明 | 必须匹配实际图片及授权/来源描述 |

## Editorial principles

### 1. 从事件走向结构

简报不只回答“发生了什么”，还追问：

- 谁提供了基础设施与制度条件？
- 谁被看见，谁承担代价？
- 一件作品、政策或技术离开原始语境后怎样被重新解释？
- 新闻中的个体故事与长期权力结构怎样相连？

### 2. 保持来源和时间

每则故事必须保留来源名称、URL 和来源日期。若主来源与补充来源承担不同作用，应像现有条目一样注明，例如“技术信息由 IMAX 交叉核验”“州级进展补充：Associated Press”。

### 3. 不把初步研究写成最终结论

小样本临床试验、预印本、新宣布的项目和仍在变化的政策必须写出限制。例如当前第 05 期使用“10 人试验里诞生了 3 个婴儿”，标题同时明确“希望仍必须写得很小心”。

### 4. 创作角度不是事实替代品

`creatorAngle` 可以提供问题、比较和叙事入口，但不能把推测写成新闻事实。它应建立在 `happened` 与 `importance` 已说明的材料之上。

### 5. 去重声明必须可解释

页面展示 `uniqueCount` 和“与历史档案无实质重复”。不要机械依靠标题不同就声称独特；新增条目前应比较人物、事件、研究、核心论点与来源。

## Fact, analysis and framing

| 层级 | 例子 | 写作要求 |
| --- | --- | --- |
| 可核验事实 | 展览日期、机构、论文样本量、政策覆盖州数 | 与来源和日期一致 |
| 背景解释 | 为什么 IMAX 胶片基础设施稀缺 | 说明证据或清楚标为解释 |
| 编辑判断 | “制度如何进入身体与公共生活” | 可作为主题，但不能替代逐条证据 |
| 创作建议 | 从“女性媒介”偏见切入水彩史 | 明确是选题角度，不冒充学术共识 |

## Content that agents must not rewrite casually

- 三条长期主题线索及其具体范围。
- 每期 5 则的编辑节奏。
- “发生了什么 / 为什么重要 / 可创作角度 / Fact File”的结构。
- 已发布条目的来源、日期、数据和限定语。
- 女性主义与制度分析不是可被“中性化”删除的附加标签。
- 天文学和地理学当前明确属于社会科学栏目在本站的编辑范围；不要擅自移除。
- `uniqueCount` 不得在未检查历史条目的情况下随意提高。

## Creator choices, not automatic defects

- 同一则故事可以拥有多个交叉类别，这不是标签不规范。
- 标题可以提出解释性问题，而不是统一成通讯社标题。
- 每期主题会把不同领域放进同一结构问题中，这种跨学科组织是项目核心。
- 图片与 Fact File 都是内容的一部分，不只是卡片装饰。
