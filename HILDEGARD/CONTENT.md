# Hildegard content guide

- Last audited: 2026-07-21
- Project: `时间的女儿 · 002`
- Primary entry: `HILDEGARD/index.html`

## Project theme

网站以宾根的希尔德加德（Hildegard of Bingen, 1098–1179）为中心。核心判断是：她不是"中世纪修道院里另一位圣女"，也不是被现代话语快速消费的"新时代女神"，而是一位在最狭窄的社会通道内建立起完整思想、音乐与语言宇宙的**谦卑的反叛者**。

网站的叙事原则已经写在页面中：

> 她的一生，如同一座在高墙之内建立起来的丰饶花园。

叙事视角引自 TSRat 稿件《谦卑的反叛者：宾根的希尔德加德》，属于**叙事框架**，不代表学界或教会官方定论。

## Target reader

- 对女性史、思想史、中世纪史和宗教史感兴趣的中文读者。
- 通过短视频、流行改编或"新时代"话语首次接触 Hildegard，希望进一步区分史料、学界推断与现代再造的读者。
- 关注教师、领导力、宗教共存和权力政治如何运作在一位女性身上的读者。
- 关注音乐、语言与神秘经验作为智识生产方式的读者。

## Core narrative

网站不把希尔德加德的一生写成"孤独天才/新时代先知"的故事，而是建立如下因果链：

```text
本笃会窄门
→ 尤塔的苦修传承 + 缮写室里的学识
→ 神启（先知）与学识（自然史 / 医学）双重权威
→ Viriditas 生命力哲学
→ Symphonia 与 Ordo Virtutum：以音乐重塑女性社群
→ 独立王国：迁鲁珀茨贝格 · 越级政治智慧
→ 通信欧洲权力中心 · 公开巡回布道
→ 晚年最后一战：拒绝迁尸 · 禁令 · 抗争 · 解禁
→ 沉默八百年 · 世俗世界的二次生命
→ 2012 等效封圣 + 教会圣师
```

## Chapter structure

网站严格按稿件章节顺序：

1. `00 引子（Portal）`：1163 年给腓特烈一世的信作为入口；预告她的多重身份。
2. `01 窄门（Manuscript）`：1098 出生 / 八岁入迪西博登贝格；时代地图（卡诺莎、教权 vs 皇权、女性两条被封死的路）。
3. `02 传承（Manuscript）`：尤塔·冯·斯蓬海姆；苦修 vs 姐妹情谊+学识两条道路的对照。
4. `03 修炼（Vision）`：神启路径与学识路径双列；Viriditas 概念在此正式出现。
5. Interlude：`"以过度苦修摧残身体之人，常行于愤怒中。"`
6. `04 音乐（Vision）`：格里高利圣咏 vs 希尔德加德对照；《天启交响曲》、《美德典律》。
7. `05 权威（Manuscript）`：温情领导、"弱势"策略、独立王国（1150）；通信卡（尤金三世 / 埃莉诺 / 亨利二世 / 腓特烈一世）；Editorial note 保留"性别观进步 vs 阶级观保守"张力。
8. `06 语言（Vision · Lingua Ignota）`：23 字母表示意 + 三种学界解读 + 芭芭拉·纽曼的社区密语假说 + 沃尔马的追问。
9. `07 斗争（Manuscript）`：清洁派、1160 起四次巡回布道、拒绝迁尸案、禁令、"禁止歌颂上帝，就是魔鬼的胜利"、1179.09.17 离世时间轴。
10. `08 身后（Botanical）`：沉默八百年、1243 文件失踪；Viriditas 当代回响、朱迪·芝加哥《晚宴》、Sequentia / Lingua Ignota / Hildegard von Blingin'；1998 900 周年、2012 等效封圣 + 教会圣师。
11. Coda Interlude：`"她的一生，如同一座在高墙之内建立起来的丰饶花园。"`

## Editorial boundaries

| 类型 | 例 | 呈现方式 |
| --- | --- | --- |
| 事实 (Historical fact) | 1098 出生、1141 神圣命令、1150 迁鲁珀茨贝格、1179.09.17 离世、2012 等效封圣 | 直接陈述 + 时间轴 |
| 学界解读 (Scholarly interpretation) | Lingua Ignota 的用途、封圣失败原因、Ordo Virtutum 与现代歌剧起源之说 | 用"学者认为"、多种解读并列 |
| 叙事框架 (Narrative framing) | "谦卑的反叛者"、"高墙之内的丰饶花园"、"神视获得发声许可，学识赋予力量" | 明确标为 TSRat 视角，Editorial note 中声明 |
| 视觉隐喻 (Visual metaphor) | Viriditas 圆环、宇宙轮 SVG、Lingua Ignota 字母示意 | `aria-hidden`，caption 或 marginalia 明确说明非史料图 |

不得进入网站的：

- 教会官方立场以外的"她其实是新时代女祭司"式改写。
- 把"红胡子"外号来源直接嫁接到 Hildegard 本人的判断上；相关历史信息只作为背景，不作她的语录。
- 把 `Lingua Ignota` 字母示意伪装成她当年的实际字形。
- 隐藏她的阶级保守立场，只放大她的性别观进步立场。

## Editorial rules for future edits

- 修正日期、著作名、事件年份时，必须给出可核查来源。
- 修改"性别观 vs 阶级观"张力段落属于**观点改变**，需 creator 明确授权。
- 增删章节前先读 `HANDOFF.md`；总屏数以八章为骨，不主动扩充章节量。
- 图像若从 AI 或后世作品引入，必须在 caption 中注明"现代视觉隐喻"或授权来源。
- 音频（如 Sequentia、Hildegard von Blingin' 改编）不在未取得授权前直接嵌入，只可外链。

## 2026-07-21 修订

回应用户反馈（首个部署版本的观感问题）：

- 减少每章正文段落数，删除重复陈述；把叙事扩写压缩为"论点 + 拉引句 + 名词/时间"。
- 每章加入 `.pull-quote` 拉引金句；不使用引号图片，只用金色发丝线上下夹持。
- 每章右上角有一个 96 × 96 的装饰性 SVG（`.chapter-glyph`），从五种既有 glyph 中挑选与章节主题匹配的一枚。
- 新增 `#watch` 章节：内嵌 Bilibili iframe（`BV1VE4gzNEYJ`），配备哔哩哔哩外链按钮。
- 新增 `#sources` 章节，与 HYPATIA 保持结构对齐：
  - `.source-table` 分层：材料 · 距离 · 最适合回答 · 主要限制。
  - `.reading-list` 外链：Barbara Newman / Sabina Flanagan / Mark Atherton / Benedict XVI 2012 册封文 / CANTUS / Sequentia / Lingua Ignota Wikipedia / TS鼠 B 站视频。
  - `.image-credits` 折叠说明：所有装饰图形均为本站 SVG，不使用 AI 生成的希尔德加德肖像。
