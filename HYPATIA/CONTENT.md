# Hypatia content guide

- Last audited: 2026-07-19
- Project: `时间的女儿 · 001`
Primary entry: `HYPATIA/index.html`

## Project theme

网站以希帕蒂娅（Hypatia）为中心，从她被谋杀的著名结局退回她实际生活过的世界。核心判断是：比起被现代头衔塑造成孤独科学天才或反宗教殉道者，她首先是一位杰出的教师。

她的课堂连接数学、天文学、哲学、德性训练、基督徒学生、帝国官员和城市公共生活。也正是这种跨宗教、无正式官职却拥有声望的教师权威，使她在 412—415 年亚历山大里亚的权力冲突中成为危险的象征。

网站的叙事原则已经写在页面中：

> 去定义她，远不如去接近她重要。

## Target reader

- 对女性史、哲学史、数学史和晚期古代感兴趣的初学者。
- 通过《城市广场》（Agora, 2009）、Carl Sagan 或大众科学叙事认识她，希望区分电影和史料的读者。
- 关注教师、知识传播、宗教共存和城市政治的读者。
- 看过“教师之死：希帕蒂娅”视频，希望慢读、比较史料和查阅来源的观众。

## Core narrative

网站不把希帕蒂娅的死亡写成“科学与宗教进行辩论，科学家因此被杀”的简单故事，而是建立以下因果链：

```text
私人和公共教学
→ 跨宗教学生网络
→ 官员与教会精英信任
→ 无官职的公共权威
→ 奥雷斯特斯与西里尔冲突中的政治谣言
→ 教师被改写成妨碍和解的敌人
→ 415 年的街头政治暴力
```

“教师之死”不只指职业身份，也指一种公开、跨宗教、不依附单一权力中心的共同学习关系被摧毁。

## Chapter structure

1. `00` 序章：从死亡退回生命；提出“去接近她”的原则和阅读地图。
2. `01` 家庭与社会：亚历山大里亚七百年历史、图书馆的制度基础与多阶段衰落、塞翁与女儿。
3. `02` 研究与思想：评注、校订、仪器知识、新柏拉图主义，以及与主流祭仪传统和基督教的具体对比。
4. `03` 教师事业：西涅修斯书信、基督徒学生、文本判断、德性教育和月经布逸闻的史料边界。
5. `04` 公共人物：没有官职的声望、塞奥菲鲁斯时代的脆弱平衡、女性进入公共空间。
6. `05` 冲突与余波：奥雷斯特斯、西里尔、街头冲突、谋杀、三种史料、责任边界和制度余波。
7. `06` 神话与象征：哲学殉道者、圣徒镜像、自由斗士、异域女神、父权制牺牲者和科学先驱。
8. `07` 最后的话：神话、史料和持续修正；“让知识流动，也让知识可被纠正”。
9. 史料与延伸阅读：材料分层、核心阅读、图片来源和授权。

## Important people and concepts

- 亚历山大里亚的塞翁（Theon of Alexandria）：数学家、天文学家和文本编辑者，希帕蒂娅的父亲。
- 昔兰尼的西涅修斯（Synesius of Cyrene）：学生，后来成为基督教主教；其书信是教学关系的一手材料。
- 奥雷斯特斯（Orestes）：罗马帝国在亚历山大里亚的行政长官，与西里尔发生权力冲突。
- 亚历山大的西里尔（Cyril of Alexandria）：412 年后主教；其阵营与政治环境高度相关，但没有直接证据证明他亲自下令谋杀。
- 苏格拉底·士林哲学家（Socrates Scholasticus）：约 439 年写作的基督教教会史家，是重建公共声望和谋杀原因的核心来源。
- 达马斯基奥斯（Damascius）：6 世纪初多神教新柏拉图主义者，保存教学逸闻，也带有敌基督教和性别偏见。
- 尼基乌的约翰（John of Nikiu）：7 世纪后期科普特主教，代表后世“巫术—政治操纵者”叙事。
- 新柏拉图主义（Neoplatonism）：太一、理智、灵魂和世界的层级传统；页面区分主流祭仪道路、基督教和从学生材料重建的 Hypatia 道路。

## Evidence levels used by the site

学术成果卡已经分为：

- 较可靠
- 学界推断
- 存在争议

例如：

| Claim | Current classification | Required boundary |
| --- | --- | --- |
| 《算术》评注 | 较可靠 | 原作散佚，无法逐段确认贡献 |
| 《圆锥曲线》工作 | 学界推断 | 更稳妥地写参与解释或校订 |
| 《天文学大成》校订 | 学界推断 | 标题语法有解释空间，不能归功整部作品 |
| 星盘与比重计知识 | 较可靠 | 她没有发明星盘 |
| 发现日心说或椭圆轨道 | 存在争议 / 无古代证据 | 只能作为电影改编讨论 |
| 第一位女性数学家 | 存在争议 | 更稳妥地称古代资料最丰富者之一 |

Agent 新增或修改学术主张时，也必须选择证据层级，不能只增加内容而不说明确定性。

## Historical fact, interpretation, framing and metaphor

### Historical fact

- 她生活在约 4—5 世纪的亚历山大里亚，并于 415 年被一群基督徒杀害。
- 她教授哲学；现存可识别学生属于基督教世界。
- 她与奥雷斯特斯交往，并在主教权与行政权冲突中被谣言锁定。
- 苏格拉底把谋杀称为政治嫉妒的结果并谴责暴力。

### Scholarly interpretation

- 她更接近普罗提诺式、重理性并与神术保持距离的道路。
- 她参与具体数学文本校订的范围。
- 西里尔承担何种政治责任。
- 护病团是否构成主要凶手群体。

### Narrative framing

- “城市本身也是主角”。
- “教师与学生网络成为新的知识制度”。
- “教师之死”意味着跨阵营共同学习的关系被破坏。
- 神话并非全都无价值，但必须允许史料不断修正。

### Visual metaphor

- `hypatia-archive-portrait.svg` 和其他创作图像不是历史肖像。
- 《城市广场》剧照用于现代叙事联想，不是现场影像。
- “走近她”AI 插画表达接近人物与持续修正，不是史料。
- `agora-hypatia-teaching.jpg` 是 2026-07-19 为修复缺图而生成的 AI 辅助现代视觉隐喻，不是《城市广场》剧照，也不证明月经布逸闻的现场细节。
- 古代柱式、陶瓶长卷和天文仪器图承担时代或概念说明，不自动证明人物曾接触该具体实物。

## Statements that must remain qualified

- 图书馆没有在 415 年随她一起被烧毁；391 年被毁的是塞拉皮翁神庙，主图书馆衰落跨越数百年。
- 她没有被证实发现日心说或椭圆轨道。
- 她没有发明星盘。
- 她不一定是第一位女性数学家。
- “现存可识别学生都是基督徒”不等于她从未教过多神教徒或犹太人；资料存在幸存偏差。
- 没有证据证明她与塞奥菲鲁斯建立正式政治同盟。
- 西里尔的政治责任与是否直接下令必须分开。
- 415 年不是古典知识传统的一日终结；哲学、数学、天文学和女性哲学家仍继续存在。

## Content that agents must not rewrite casually

- 教师身份高于“孤独天才”神话的核心判断。
- 宗教、城市政治、行政权和群众动员的复合解释。
- 对基督教与新柏拉图主义的具体差异；不能只说“都有灵魂和超越”。
- 三种谋杀史料的作者位置、价值和限制。
- 对西里尔责任的谨慎边界。
- 性别既提供结构限制，也不能把她只写成被害者。
- “去接近人物，而不是占有唯一真相”的结尾。

## Creator choices, not automatic defects

- 页面花较长篇幅讲城市、图书馆制度和哲学背景，是为了避免人物悬浮在抽象“文明冲突”中。
- 电影和后世神话没有被简单删除，而被作为记忆政治和现代入口分析。
- 页面不复现详细身体暴力，而追问暴力怎样被制度和谣言组织。
- 同一页面同时承认传统延续和具体关系断裂，这不是逻辑矛盾。
